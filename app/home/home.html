How to use the ConSpeciFix tool to identify your bacteria:
  • Prepeare your data: find your FASTA file (.fa) containing the test genome (an example from NCBI. Make sure to unzip the file before using it here!)
  • Upload your data: drag and drop your file into the field below
  • Tell us what you know: select the species set you would like to test your genome against.
  • Get Answers: we'll email you when we have some results to share with you.

Happy testing!
